# ⚡ AI-Based Smart Energy Management System

An AI-powered web application to monitor, predict, and optimize electricity consumption with interactive dashboards, intelligent forecasts, and energy-saving recommendations.

## 🚀 Project Overview

The **AI-Based Smart Energy Management System** helps users make informed decisions about energy usage by providing:
- Real-time usage tracking
- AI-based consumption predictions
- Electricity bill estimation
- Smart energy-saving recommendations

This tool is designed for households, businesses, and industries aiming for smarter, cost-effective, and sustainable energy practices.

## 🔧 Tech Stack

**Frontend**:  
- HTML, CSS, JavaScript  
- ReactJS  
- Google Looker Studio (for data visualization)

**Backend**:  
- Java Spring Boot  
- REST APIs  
- PostgreSQL (Database)

**AI & Analytics**:  
- Machine Learning model for predictions  
- Bill Estimator logic  
- Recommendation system

## 📌 Key Features

### 🔐 Authentication
- User registration and login system with personalized dashboard.

### 📊 Dashboard
- Integrated with Google Looker Studio for real-time energy reports.
- View detailed analytics of energy consumption.

### 📈 AI-Based Predictions
- Select date and hour to forecast future electricity usage.
- Helps in planning energy consumption.

### 💰 Bill Estimation
- Calculates expected electricity bill based on usage and tariff.
- Avoid surprise bills and budget efficiently.

### 💡 AI Recommendations
- Personalized suggestions to reduce energy waste.
- Promotes sustainability through smart usage tips.

## ⚙️ How to Run

### Backend (Spring Boot):
1. Import the project into Eclipse or IntelliJ.
2. Configure PostgreSQL database in `application.properties`.
3. Run the Spring Boot application.

### Frontend:
1. Navigate to the frontend folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the development server.
